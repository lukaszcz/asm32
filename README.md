ASM32 is an educational assembler for a subset of 32bit x86 instructions. It produces Linux ELF executables.

Requirements
------------
* Linux
* C compiler

Usage
-----
* Compilation: `make`
* Tests: `make test`
* Invocation: `asm32 program.asm`
